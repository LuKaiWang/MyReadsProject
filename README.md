# MyReads Project 项目简介

该项目主页显示3层书架，分别为Currently Reading，Want to Read和Read，每层书架都包含相应图书，每本图书都可选择不同书架显示。查找页面可显示按限定关键词查找的图书，并将图书加入主页不同书架。所有图书在主页和查找页面都保持统一状态。


## 安装说明

* 安装所有项目依赖包命令 `npm install`
* 启动项目命令 `npm start`
